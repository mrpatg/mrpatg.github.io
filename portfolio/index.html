Portfolio

Below you will find a sampling of some websites that I have built or worked on with a brief synopsis of what my role was, and what technologies were used.

Bluemagnolia Home
Website - Uptime Report

Bluemagnolia Home is a boutique/decore shop. Client had existing business but wanted to transform with entirely new branding/ident. Tasks included:

  • Creating Logo and branding material
  • Register domain and establish Wordpress site
  • Gather content from client including photos
  • Create unified gmail account to point mailgun routes
  • Setup and configure Facebook, Instagram, Twitter, and Google Places
  • Created new Google Places entry with new business information
  • Setup MailChimp account and linked website signup form to the mailing list
  • Assessed client needs and created layout/content structure that best suited their specific needs

Technologies used in this project

  • Namecheap
  • Cloudflare
  • DigitalOcean
  • Server Pilot
  • PHP7 + Wordpress
  • Let's Encrypt SSL Certificate
  • Modified Uncode Wordpress Theme
  • Gravity Forms with Mailchimp Addon
  • Mailgun
  • Google Analytics
  • Google Places
  • Uptime Robot
